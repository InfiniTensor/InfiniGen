## Get started 

### Make Commands
- `make build`: Builds the projects;
- `make test TESTCASE=$(TESTCASE) PLATFORM=(CUDA/BANG)`: Test the code generation and verify its correctness on the hardware accelerator.
- `make tests`: Run all tests.
